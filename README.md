# DingDingPod

## Features

- 防撤回
- 自动抢红包
- 抢自己
- 延迟秒数
- 修改步数

## Installation

DingDingPod is available through [MonkeyDevSpecs](https://github.com/AloneMonkey/MonkeyDevSpecs.git). To install
it, simply add the following line to your Podfile:

```ruby
pod 'DingDingPod'
```
